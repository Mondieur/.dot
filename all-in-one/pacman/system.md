# Linux OS
1. Create file `touch bash.sh`
2. Remove file `rm bash.sh`
3. Rename file `mv bash.sh file.sh`
4. Move file (to parent folder) `mv file.sh ../`
5. Move nested folder: `mv -r ./nested ../../home`

6. Disk usage: `du -sh`
